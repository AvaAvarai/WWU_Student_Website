Parallel Coordinates plots of the hydrogen bonds lost for each amino acid inserted into the protein.

Each polyline is a loss value on each protein-protein interface for one insertion of the amino acid.

Values are produced computationally by the Rosetta software.

Since Rosetta is non-deterministic, multiple runs are shown for each amino acid insertion.
