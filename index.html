Visual Analysis of Protein Insertion Hydrogen Bond Loss

Parallel Coordinates plots of the hydrogen bonds lost for each amino acid inserted into the protein.

Each polyline is a 196-dimensional vector of loss values on each of the 196 protein-protein interfaces for one insertion of the amino acid.

Values are produced computationally by multiple runs of the Rosetta software.

Since Rosetta is non-deterministic, multiple runs are shown for each amino acid insertion, some values also require multiple runs of Rosetta to produce a non-empty value, some empty values remain.

Empty values are represented by an added box on the Parallel Coordinated axis for that interface which that insertion polyline crosses into.

Protein: 1brs

Amino Acids:

  • Alanine (A) 1brs amino acid a
  • Cysteine (C) 1brs amino acid c
  • Aspartic Acid (D) 1brs amino acid d
  • Glutamic Acid (E) 1brs amino acid e
